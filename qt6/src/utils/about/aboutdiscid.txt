License 2.1 or later</a>.

It reads a CD's table of contents (TOC) and generates an identifier which can be used to lookup the CD at MusicBrainz.

The library also provides <a href="https://gnudb.org/" target="_blank">FreeDB</a> disc IDs, and MCN + ISRCs, if available.
